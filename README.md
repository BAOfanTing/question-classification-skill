# 公共基础知识真题分类整理

> 将事业单位/公务员《公共基础知识》考试套卷按科目智能分类，一键生成带目录的 Word 文档。

## 功能

输入任意数量的《公共基础知识》真题套卷（PDF），自动：

1. **提取题目** — 从 PDF 提取文字内容
2. **按科分类** — 将每道题归入以下 **14 个科目**：

| # | 分类 | 核心范围 | 典型关键词 |
|---|------|----------|------------|
| 1 | **刑法** | 犯罪构成、刑罚、刑事诉讼 | 故意杀人、抢劫、盗窃、管制、拘役、缓刑、正当防卫 |
| 2 | **民法** | 民事主体、物权、合同、婚姻继承 | 合同法、婚姻、继承、侵权、物权、诉讼时效 |
| 3 | **宪法** | 国家制度、国家机构、基本权利 | 人大、国务院、选举、基本权利、立法法、特区制度 |
| 4 | **行政法与诉讼法** | 行政行为、行政处罚、行政诉讼 | 行政处罚、许可、强制、复议、诉讼、国家赔偿、公务员 |
| 5 | **商法与经济法** | 公司、劳动、消费者权益、竞争 | 劳动合同、试用期、消费者权益、反不正当竞争、社保 |
| 6 | **管理** | 管理原理、行政管理、组织行为 | 决策、组织、领导、控制、激励理论、绩效评估 |
| 7 | **公文** | 公文种类、格式、行文规则 | 通知、请示、报告、函、纪要、发文字号、版头 |
| 8 | **经济** | 微观/宏观经济学、财政政策 | GDP、通货膨胀、货币政策、供需弹性、边际效用 |
| 9 | **马哲** | 唯物论、辩证法、认识论、历史唯物主义 | 唯物主义、唯心主义、矛盾、实践、真理、生产力与生产关系 |
| 10 | **毛思想+近代史** | 毛泽东思想、新民主主义革命、近代史 | 毛泽东、新民主、井冈山、长征、三大改造、辛亥革命 |
| 11 | **中特+习思想** | 邓小平理论、三个代表、科学发展观、习近平新时代 | 习近平、改革开放、一带一路、四个全面、共同富裕 |
| 12 | **人文** | 文学、历史、艺术、传统文化 | 唐诗、宋词、京剧、名著、历史人物、传统节日、二十四节气 |
| 13 | **时政** | 近期国内外重大新闻、政策文件 | 领导人讲话、重要会议、科技成就、新法实施 |
| 14 | **地理科技** | 自然科学、生活常识、计算机 | 物理化学生物、地理天文、计算机操作、安全用药 |

3. **智能二次分类** — 自动从混合科目（如"中特"）中拆分出马哲题、毛思想题
4. **格式统一** — 每题标注 `(年月 地点)` 前缀，按时间排序
5. **生成 Word** — 带目录、页码、分科排列、答案横排在模块底部

## 目录结构

```

│   └── generate_final.py              # 合并生成 Word 的脚本

├── 套卷/                          # 放入待整理的 PDF 真题

```

## 使用方法

### 方式一：通过 Claude Code 调用（推荐）

在项目目录下运行：

```
/question-classification --source 套卷路径 --output 输出路径
```

不传参数则使用默认路径。

### 方式二：手动执行

```bash
# 1. 提取 PDF 文本
cd 套卷/
for f in *.pdf; do
  pdftotext -layout -enc UTF-8 "$f" "../extracted_txt/${f%.pdf}.txt"
done

# 2. 用 AI 逐套分类提取题目（按 skill 中的分类规则）

# 3. 合并生成 Word
cd ../分类题目
python generate_final.py
```

### 方式三：作为独立脚本

```bash
python generate_final.py
```

## 依赖

- Python 3.7+
- `python-docx`：`pip install python-docx`
- `pdftotext`（poppler-utils）：用于 PDF 文本提取

## 输出格式

Word 文档结构：

```
封面标题
目录（在 Word 中更新域后自动生成）

刑法
  ├── 单选 1.(2023年8月 宁波北仑)...
  ├── 多选 ...
  ├── 判断 ...
  └── 答案 1.A  2.B  3.C ...

民法
  └── ...

...（共14个科目）
```
