# 公共基础知识真题分类整理

> 将事业单位/公务员《公共基础知识》考试套卷按科目智能分类，一键生成带目录的 Word 文档。

## 功能

输入任意数量的《公共基础知识》真题套卷（PDF），自动：

1. **提取题目** — 从 PDF 提取文字内容
2. **按科分类** — 将每道题归入以下 **14 个科目**：

| 法律类 | 管理公文类 | 政治理论类 | 综合类 |
|--------|-----------|-----------|--------|
| 刑法 | 管理 | 马哲 | 人文 |
| 民法 | 公文 | 毛思想+近代史 | 时政 |
| 宪法 | | 中特+习思想 | 地理科技 |
| 行政法与诉讼法 | | | |
| 商法与经济法 | | | |

3. **智能二次分类** — 自动从混合科目（如"中特"）中拆分出马哲题、毛思想题
4. **格式统一** — 每题标注 `(年月 地点)` 前缀，按时间排序
5. **生成 Word** — 带目录、页码、分科排列、答案横排在模块底部

## 目录结构

## 使用方法

### 方式一：通过 Claude Code 调用（推荐）

在项目目录下运行：

```
/question-classification --source 套卷路径 --output 输出路径
```

不传参数则使用默认路径。

### 方式二：手动执行

```bash
# 1. 提取 PDF 文本
cd 套卷/
for f in *.pdf; do
  pdftotext -layout -enc UTF-8 "$f" "../extracted_txt/${f%.pdf}.txt"
done

# 2. 用 AI 逐套分类提取题目（按 skill 中的分类规则）

# 3. 合并生成 Word
cd ../分类题目
python generate_final.py
```

### 方式三：作为独立脚本

```bash
python generate_final.py
```

## 分类规则概览

### 14 个科目的核心判断标准

| 分类 | 判断依据 |
|------|----------|
| **刑法** | 犯罪（故意杀人/抢劫/盗窃）、刑罚（管制/拘役/缓刑）、正当防卫 |
| **民法** | 合同、婚姻、继承、物权、侵权、诉讼时效 |
| **宪法** | 国家机构（人大/国务院）、基本权利、立法法、选举制度 |
| **行政法与诉讼法** | 行政处罚、行政许可、行政复议、行政诉讼、国家赔偿 |
| **商法与经济法** | 劳动合同、消费者权益、不正当竞争、社会保险 |
| **管理** | 决策、组织、领导、激励理论、绩效评估、行政管理 |
| **公文** | 15 种公文种类、格式规范、行文规则、处理流程 |
| **经济** | 微观（供需/弹性）、宏观（GDP/通胀）、财政货币政策 |
| **马哲** | 唯物主义/唯心主义、辩证法、认识论、历史唯物主义（**仅限明确哲学题**） |
| **毛思想+近代史** | 毛泽东、新民主主义革命、中共党史、鸦片战争—三大改造 |
| **中特+习思想** | 邓小平理论、三个代表、科学发展观、习近平新时代、改革开放 |
| **人文** | 文学（唐诗/宋词/名著）、历史、艺术（京剧/书法）、传统文化 |
| **时政** | 近期新闻（具体日期）、领导人活动、重要会议政策 |
| **地理科技** | 理化生、地理天文、计算机、生活常识、环保 |

> 详细分类规则见 `.claude/skills/question-classification.md`

## 依赖

- Python 3.7+
- `python-docx`：`pip install python-docx`
- `pdftotext`（poppler-utils）：用于 PDF 文本提取

## 输出格式

Word 文档结构：

```
封面标题
目录（在 Word 中更新域后自动生成）

刑法
  ├── 单选 1.(2023年8月 宁波北仑)...
  ├── 多选 ...
  ├── 判断 ...
  └── 答案 1.A  2.B  3.C ...

民法
  └── ...

...（共14个科目）
```
