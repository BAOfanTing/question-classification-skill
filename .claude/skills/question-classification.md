# Question Classification Skill

从指定文件夹的套卷PDF（或已提取的TXT）中提取题目，按14个科目智能分类，生成带目录和页码的 Word 文档。

## 调用方式

```
/question-classification --source E:/考试/真题 --output E:/考试/分类汇总.docx
/question-classification   # 使用项目默认路径
```

## 参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--source` | 套卷PDF所在文件夹 | 项目 `到时候打印的/套卷` |
| `--output` | 输出Word路径 | `{source}/../分类汇总.docx` |

---

## 详细分类规则（14科）

分类优先级：先判断是否属于特殊类别（马哲→毛思想+近代史→行政法→商经法），再按常规归类。**同一道题只能归入一个类别**，优先归入最匹配的类别。

### 1. 刑法

**内容范围**：犯罪构成、刑罚种类、具体罪名、刑事诉讼程序

**典型信号词**：
- 犯罪：故意犯罪、过失犯罪、犯罪预备、犯罪未遂、犯罪中止、共同犯罪、主犯、从犯、胁从犯
- 刑罚：管制、拘役、有期徒刑、无期徒刑、死刑、罚金、剥夺政治权利、没收财产、缓刑、假释
- 罪名：抢劫罪、盗窃罪、贪污罪、受贿罪、行贿罪、故意伤害罪、交通肇事罪、危险驾驶罪、强奸罪、绑架罪、诈骗罪
- 程序：刑事诉讼法、立案、侦查、提起公诉、审判、上诉、抗诉
- 原则：罪刑法定、罪刑相适应、适用刑法平等、从旧兼从轻
- 制度：自首、立功、正当防卫、防卫过当、紧急避险

**典型题型示例**：
```
(2023年8月 宁波北仑) 关于缓刑，下列说法中错误的是（ ）
(2022年11月 宁波市属) 王某酒后扬言要杀死厂长...厂长的行为属于（ ）
```

**与其他类别的区分**：
- vs 宪法：刑法是具体部门法，涉及具体罪名和刑罚；宪法是国家根本制度
- vs 行政法：刑法的处罚是刑罚（管制拘役有期徒刑等）；行政法是行政处罚（罚款拘留等）

### 4. 行政法与诉讼法

**内容范围**：行政行为、行政处罚、行政许可、行政强制、行政复议、行政诉讼、国家赔偿

**典型信号词**：
- 行政主体、行政机关、行政行为、具体行政行为、抽象行政行为
- 行政处罚：警告、罚款、没收、责令停产停业、暂扣吊销许可证、行政拘留
- 行政许可、行政审批
- 行政强制措施、行政强制执行
- 行政复议、复议机关、复议范围
- 行政诉讼、原告、被告、举证责任
- 国家赔偿、行政赔偿、赔偿范围
- 公务员法、公务员管理
- 政府信息公开、信息公开条例
- 治安管理处罚法（区别于刑法）

**与其他类别的区分**：
- vs 刑法：行政处罚（罚款拘留）≠ 刑罚（管制有期徒刑）；行政违法≠犯罪
- vs 宪法：行政法是具体部门法，涉及政府行政行为；宪法是国家根本制度
- 如果题中出现"行政处罚""行政复议"等字眼，优先归入行政法

**典型题型示例**：
```
(2023年3月 宁波北仑) 下列有关人民法院的说法，错误的是（ ）
——但注意人民法院属于宪法（国家机构），如果是行政诉讼才归行政法
```

### 9. 马哲（马克思主义哲学）

**内容范围**：唯物论、辩证法、认识论、历史唯物主义。**仅限明确的哲学题，需题干从哲学角度提问。**

**核心判断标准**：题干或选项中明确出现"哲学""哲学道理""哲学原理""唯物""唯心""辩证法""认识论"等术语，或涉及典型的哲学范畴讨论。

**详细关键词**：
- **唯物论**：物质、意识、物质决定意识、意识能动作用、客观规律、主观能动性、实事求是、一切从实际出发
- **辩证法**：矛盾、对立统一、量变与质变、否定之否定、普遍联系、变化发展、内因外因、矛盾普遍性与特殊性、主要矛盾与次要矛盾、矛盾主要方面
- **认识论**：实践、认识、感性认识、理性认识、真理、谬误、实践是检验真理的唯一标准、认识的过程
- **历史唯物主义**：社会存在、社会意识、生产力、生产关系、经济基础、上层建筑、社会基本矛盾、人民群众
- **哲学流派**：唯物主义、唯心主义（主观/客观）、古代朴素唯物主义、近代形而上学唯物主义、辩证唯物主义、形而上学
- **典型句式**："从哲学上看""体现了什么哲学道理""包含的哲学原理""上述材料蕴含的哲学道理"

**⚠️ 排他规则**：如果题干是关于经济的但用了哲学词，不要归入马哲。只有题干本质上是哲学问题才归入马哲。

**典型题型示例**：
```
(2021年11月 宁波北仑) 下列属于主观唯心主义观点的是（ ）
(2022年2月 宁波镇海) 中国和日本两国关系的发展...体现了什么哲学道理（ ）
(2022年11月 宁波市属) "刻舟求剑"是脱离物质谈运动的主观唯心主义。（ ）
```

**不要归入马哲的例子**：
```
"创新是引领发展的第一动力"——这是习思想/中特
"社会主义的本质要求是共同富裕"——这是中特
"我国社会主要矛盾已经转化"——这是中特
```

### 10. 毛思想+近代史

**内容范围**：毛泽东思想的形成与发展、新民主主义革命理论、中国近代史重要事件

**详细关键词**：
- **毛泽东与毛泽东思想**：毛泽东、毛选、毛泽东思想、活的灵魂、实事求是、群众路线、独立自主
- **新民主主义革命**：新民主主义、总路线、三大法宝（统一战线、武装斗争、党的建设）、农村包围城市、工农武装割据
- **重要会议**：中共一大、遵义会议、八七会议、古田会议、瓦窑堡会议、洛川会议、中共七大、七届二中全会
- **革命历程**：五四运动、中国共产党成立、第一次国共合作、北伐、土地革命、长征、抗日战争、百团大战、解放战争
- **近代史**：鸦片战争、太平天国、洋务运动、戊戌变法、义和团、辛亥革命、北洋政府
- **社会主义革命**：过渡时期总路线、三大改造、社会主义改造
- **党的建设**：延安整风、三大作风、两个务必

**与中特的区分**：
- 毛思想+近代史：1949年前后的革命与建设理论，近代史事件

- 模糊地带：如果题同时涉及毛和中特，看主要倾向

**典型题型示例**：
```
(2023年3月 宁波北仑) 中共七大的政治路线...这次党的代表大会是（ ）

```

**典型题型示例**：


### 13. 时政

**内容范围**：近期国内外重大新闻、政策文件、重要会议、领导人讲话

**典型特征**：题中出现具体日期、近期事件、领导人活动、重要政策发布

**详细关键词**：
- 近期年份（当年或前1-2年）具体日期
- 领导人考察、讲话、指示、贺信、回信
- 重要会议：两会、中央经济工作会议、中央农村工作会议、党代会、全会
- 政策文件：政府工作报告、十四五规划、远景目标
- 外交活动：出访、会见、国际会议、主场外交
- 科技成就：航天发射（神舟、嫦娥、天问）、重大科技突破
- 法律法规新颁：新法实施、法律修订
- 国际热点：国际会议、全球倡议、双边关系

**典型题型示例**：
```
(2023年6月 宁波镇海) 2023年5月，第133届（ ）线下展在广州圆满落幕
(2023年8月 宁波北仑) 2023年7月28日，中国选手（ ）打破世界纪录...
```

**区分规则**：

## 工作流程

### 第1步：提取PDF文本
```bash
cd "{source}"
for f in *.pdf; do
  pdftotext -layout -enc UTF-8 "$f" "../extracted_txt/${f%.pdf}.txt"
done
```

### 第2步：Agent逐套分类提取
对每个txt文件，启动agent阅读全文，按上述14科分类规则提取题目。
每道题格式：`(YYYY年M月 地点) 题干...` + 选项 + `答案：X`

### 第3步：合并生成Word文档
运行 `scripts/generate_final.py`：
1. 读取所有分类后的md文件
2. 智能二次分类
3. 按日期排序，分单选/多选/判断
4. 生成Word（目录域 + 各科内容 + 答案横排 + 页码）

### 第4步：输出
告知用户输出路径，提示在Word中更新目录域。
